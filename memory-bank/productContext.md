# 产品上下文

## 目标用户
- Cocos Creator 游戏开发者
- 需要高性能游戏逻辑的团队
- 希望使用 ECS 架构模式的开发者

## 使用场景

### 场景 1: 高性能游戏
- 大量游戏对象（如弹幕游戏、RTS 游戏）
- 需要优化的系统执行
- 数据导向设计

### 场景 2: 复杂游戏逻辑
- 需要灵活的组件组合
- 系统解耦
- 易于扩展和维护

### 场景 3: 团队协作
- 清晰的架构边界
- 类型安全的 API
- 易于理解和维护

## 核心价值

1. **性能**
   - 数据局部性优化
   - 批量处理能力
   - 内存效率

2. **可扩展性**
   - 组件化设计
   - 系统解耦
   - 易于添加新功能

3. **开发体验**
   - 与 Creator 无缝集成
   - TypeScript 类型支持
   - 清晰的 API 设计

## 成功指标

- 支持 1000+ Entity 流畅运行
- API 易于学习和使用
- 与 Creator 原生性能相当或更好
- 完整的文档和示例
